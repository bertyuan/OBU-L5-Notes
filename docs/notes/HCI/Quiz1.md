# Quiz 1

---

1. Which of the following best describes the main focus of Human-Computer Interaction (HCI)?<br>
(A) How computers communicate with other devices<br>
(B) How humans interact with computer systems effectively and efficiently<br>
(C) How to increase the speed of computer networks<br>
(D) How software is programmed using different languages<br>

Answer: (B) How humans interact with computer systems effectively and efficiently<br>

---

2. In HCI, which term refers to the design of systems that are easy for people to use?<br>
(A) Complexity<br>
(B) Usability<br>
(C) Functionality<br>
(D) Productivity<br>

Answer: (B) Usability<br>

---

3. Which of the following is an example of a user interface?<br>
(A) A printer’s internal circuit<br>
(B) A website’s navigation menu<br>
(C) A computer’s cooling fan<br>
(D) A Wi-Fi router’s antenna<br>

Answer: (B) A website’s navigation menu<br>

> [!TIP]
> antenna n. 触须，触角; <美> 天线; 感觉, 直觉<br>
> 这里指 Wi-Fi 路由器的天线<br>

---

4. What is the purpose of HCI design principles?<br>
(A) To make systems harder to use<br>
(B) To guide designers in creating efficient, user-friendly interfaces<br>
(C) To speed up computer processing<br>
(D) To remove all visual elements from interfaces<br>

Answer: (B) To guide designers in creating efficient, user-friendly interfaces<br>

---

5. Which of the following actions demonstrates good feedback in a user interface?<br>
(A) The system ignores user input<br>
(B) A progress bar showing file upload completion<br>
(C) An application closing suddenly without notice<br>
(D) A button that performs no visible action<br>

Answer: (B) A progress bar showing file upload completion<br>

---

6. In HCI, learnability refers to:
(A) How quickly a new user can learn to use a system<br>
(B) How fast the system processes information<br>
(C) How well a user can forget previous actions?<br>
(D) How complex the system’s backend is<br>

Answer: (A) How quickly a new user can learn to use a system<br>

---

7. What is user-centered design in HCI?<br>
(A) A design process focused on technical performance<br>
(B) A design approach that prioritizes user needs and feedback<br>
(C) A method that ignores user testing<br>
(D) A process that focuses only on developers’ preferences<br>

Answer: (B) A design approach that prioritizes user needs and feedback<br>

---

8. Which of the following is a benefit of applying HCI principles in software design?<br>
(A) Increased system errors<br>
(B) Higher user satisfaction and productivity<br>
(C) More complex navigation structures<br>
(D) Less accessibility for users<br>

Answer: (B) Higher user satisfaction and productivity<br>

---

9. In HCI, feedback helps users:<br>
(A) Understand the result of their actions within the system<br>
(B) Access hidden system files<br>
(C) Write better code<br>
(D) Increase system speed<br>

Answer: (A) Understand the result of their actions within the system<br>

---

10. Why is consistency important in user interface design?<br>
(A) It forces users to relearn new layouts frequently<br>
(B) It helps users predict system behavior and reduces confusion<br>
(C) It makes interfaces appear more random<br>
(D) It prevents users from memorizing actions<br>

Answer: (B) It helps users predict system behavior and reduces confusion<br>
